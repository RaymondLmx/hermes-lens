# Changelog

## 0.1.0

- Initial usable Hermes Lens release.
- FastAPI backend for session index, event replay, SSE streaming, and media.
- React/Vite frontend with Activity, Errors, Tools, and Debug views.
- Read-only Hermes exporter plugin for JSONL event streams.
- Media reference support and allowlisted media endpoint.
- Four themes: Hermes Dark, Hermes Light, VS Code Dark, VS Code Light.
- Settings drawer for timeline/debug display preferences.
